<summary><b>AutoReadme</b> &nbsp;·&nbsp; AI README Generator</summary>
<br/>

> Paste any GitHub repo URL and get a complete, production-grade README in under 15 seconds. Fetches real repo data — languages, file tree, dependencies, topics — and feeds it into Groq's LLaMA 3.3 70B for a README that reads like a human wrote it. Word-by-word streaming animation, one-click download, 5 built-in server Groq keys with rotation (72,000 free generations/day), a full Netlify serverless backend, and a CI/CD pipeline with 20 automated tests scanning for leaked secrets on every push.

**Stack:** Vanilla JS · Netlify Serverless Functions · Groq LLaMA 3.3 · GitHub REST API · GitHub Actions CI/CD
